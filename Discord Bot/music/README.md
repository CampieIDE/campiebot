# Music Files Directory

Place your music files (MP3, WAV, etc.) in this directory to play them with the `/play` command.

**Usage:**
- Place audio files in this folder
- Use `/play <filename>` to play them (e.g., `/play song.mp3`)

**Supported formats:**
- MP3
- WAV
- OGG
- Other formats supported by ffmpeg

**Note:** Make sure the bot has proper permissions to read files from this directory.
